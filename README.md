## üöÄ Getting Started

### Prerequisites

- Python 3.8 or higher
- pip (Python package installer)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/psyll0n/python-training.git
cd python-training
```

2. Install required dependencies (varies by project):
```bash
pip install -r requirements.txt  # If available
```

### Usage

Navigate to any directory and run the Python scripts:
```bash
cd 01_fundamentals/conditions
python conditions_example.py
```

## üìñ Learning Path

### For Beginners:
1. Start with **01. Fundamentals** - Learn basic syntax, conditions, loops, and functions
2. Move to **02. Data Structures** - Master lists, dictionaries, and comprehensions
3. Practice with **11. Exercises** - Apply your knowledge with practical exercises
4. Try **10. Projects/Simple Games** - Build fun projects to solidify concepts

### For Intermediate Learners:
1. Explore **03. Advanced Concepts** - OOP, decorators, async programming
2. Study **04. Standard Library** - Learn to use Python's powerful built-in tools
3. Work with **05. File and Data** - Data processing and analysis
4. Build with **06. Web and APIs** - Create real-world applications

### For Advanced Learners:
1. Dive into **07. Frameworks** - Web application development
2. Explore **08. Cloud Services** - Cloud integration and deployment
3. Master **09. Algorithms** - Efficient algorithm implementations
4. Create complex applications using multiple concepts together
